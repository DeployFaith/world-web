Hermes Internet

Hermes_OS

Welcome to the bundled local internet for Hermes_OS: fast, in-process pages designed for the OS, its apps, and future in-world experiences.

Home Interactive JS Games
Local by design. Hermes Internet pages are bundled with Hermes_OS and load directly in the Browser. They are not live external websites, do not require Docker, and do not depend on a localhost sidecar.
▦

Apps

Launch built-in tools from the dock or command surfaces. Hermes_OS apps are native, themed, and available without leaving the desktop.

▣

Files

Browse the Hermes_OS virtual filesystem, open documents, organize folders, and keep player or agent-created artifacts in one place.

⌁

Terminal

Use the local virtual shell for filesystem commands, app actions, and Hermes Gateway-backed conversations through the production Terminal.

✦

Hermes Chat

Talk with Hermes inside the OS using the dedicated chat interface, built on the same visual system as the rest of Hermes_OS.

⚙

System Settings

Review Gateway and MCP status, adjust appearance, and manage the OS environment from a focused settings app.

◎

Browser / Hermes Internet

The Browser opens bundled Hermes Internet domains like home.hermes. Real Internet browsing is intentionally out of scope for this foundation sprint.

What comes next?

This first site establishes the in-process resolver, registry, and document loader. Future Hermes Internet sites can be added as bundled content without adding a server or external WorldWeb repository.
